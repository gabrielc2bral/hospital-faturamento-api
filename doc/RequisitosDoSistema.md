# Requisitos

## 1. Cadastro

- O sistema deve permitir o cadastro de pacientes.
- O sistema deve permitir o cadastro de médicos.
- O sistema deve permitir associar procedimentos a médicos.
- O sistema deve permitir o cadastro de procedimentos.
- O sistema deve permitir o cadastro de consultas associando um paciente a um médico.

## 2. Consulta

- O sistema deve permitir filtrar médicos por especialidade.
- O sistema deve permitir buscar pacientes cadastrados.
- O sistema deve permitir buscar médicos cadastrados.
